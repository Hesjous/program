Алгоритм построения:
Исходный треугольник задаётся тремя координатами вершин:
my_points = [[-200, -100], [0, 200], [200, -100]] — равносторонний треугольник, ориентированный вершиной вверх.
Функция draw_triangle(points, color, my_turtle):
Перемещает черепаху к первой вершине.
Рисует контур треугольника по трём точкам.
Заливает его указанным цветом.
Функция get_mid(p1, p2):
Вычисляет координаты середины отрезка между двумя точками — ключевой шаг для деления треугольника.
Рекурсивная функция sierpinski(points, degree, my_turtle):
Базовый случай: если degree == 0, рисуется только текущий треугольник.
Рекурсивный шаг: вычисляются три середины сторон, образующие внутренний треугольник, и рекурсивно вызывается для трёх внешних треугольников.
На каждом уровне глубины (degree) применяется свой цвет из списка colormap, что создаёт радужный эффект.
Настройки черепахи и окна:
Скорость рисования установлена на максимальную (speed(0)).
Черепаха скрыта (hideturtle()), чтобы не мешать виду.
Фон окна чёрный (bgcolor("black")) — для контраста с яркими цветами.
Окно закрывается по клику (exitonclick()).
Глубина рекурсии: degree = 5 — обеспечивает достаточную детализацию без перегрузки системы.
